# OZON Brand Guard

🛡️ Расширение для браузера, которое автоматически защищает ваши бренды от подделок на маркетплейсе OZON.

## Возможности

- **Сканирование продавцов** — автоматический парсинг страницы «Продавцы бренда»
- **Автопоиск брендов** — автоматическое определение брендов на странице с накоплением по страницам
- **Whitelist** — разрешённые продавцы по имени или ИНН не получают жалобы
- **Фильтр по странам** — определение нарушителей по стране (CN, TR, KR, IN)
- **Авто-жалобы** — заполнение текста, прикрепление торгового знака, отправка
- **Мульти-бренд** — разные шаблоны и файлы для каждого бренда, красная подсветка если файл не прикреплён
- **История жалоб** — до 5000 записей в таблице (дата, продавец, ИНН, бренд, страна, статус)
- **Экспорт жалоб** — CSV, Excel (.xls), TXT с полной информацией о нарушителях
- **Технические логи** — отдельная вкладка с DIAG-логами в реальном времени
- **Расписание** — автозапуск через Chrome Alarms API
- **Dry Run** — тестовый режим без отправки жалоб
- **Экспорт/импорт** настроек

## Совместимость

Работает во всех Chromium-браузерах:
- Яндекс Браузер
- Google Chrome
- Microsoft Edge
- Opera / Brave

## Установка

1. Скачайте папку `ozon-brand-guard`
2. Откройте в браузере `chrome://extensions/` (или `browser://extensions/` в Яндекс Браузере)
3. Включите **Режим разработчика**
4. Нажмите **Загрузить распакованное расширение**
5. Выберите папку `ozon-brand-guard`

## Быстрый старт

1. Откройте [Продавцы бренда](https://seller.ozon.ru/app/brand/sellers) в OZON Seller
2. Нажмите на иконку расширения (синий щит)
3. Вкладка **Бренды** → «Найти бренды на странице» → отметьте свои
4. Добавьте текст жалобы и файл торгового знака для каждого бренда
5. Вкладка **Главная** → «Сканировать + жалобы» → задержка 15-30с → **Сканировать**

> Подробная инструкция: [GUIDE.md](GUIDE.md)

## Структура

```
ozon-brand-guard/
├── manifest.json           # Manifest V3
├── popup/                  # UI расширения (6 вкладок)
├── content/                # Content script (работает на странице OZON)
├── background/             # Service Worker (планировщик, логирование)
├── options/                # Страница расширенных настроек
├── assets/                 # Иконки (SVG щит + PNG)
└── _locales/               # Локализация
```

## Вкладки расширения

| Вкладка | Описание |
|---------|----------|
| **Главная** | Управление, режим, задержка, расписание |
| **Бренды** | Автопоиск + список брендов с файлами |
| **Whitelist** | Разрешённые продавцы + фильтр по странам |
| **Настройки** | Текст жалобы, Dry Run, уведомления, данные |
| **Жалобы** | Таблица истории (5000) + экспорт CSV/Excel/TXT |
| **Инфо** | Краткая инструкция и совместимость |

## Настройки

### Бренды
Добавьте бренды через вкладку «Бренды». Для каждого можно задать:
- Название (как на OZON)
- Текст жалобы (или используется текст по умолчанию)
- Файл торгового знака (PDF, JPG, PNG)

> Бренды без прикреплённого файла подсвечиваются красным.

### Whitelist
Добавьте разрешённых продавцов по имени или ИНН через вкладку «Whitelist».

### Фильтр по странам
Включите/выключите фильтрацию по стране продавца. По умолчанию CN (Китай) помечен как нарушитель.

### Задержка
Между жалобами выдерживается пауза 10-300 секунд (настраивается, рекомендуется 15-30с) + случайный джиттер 0-5 секунд.
