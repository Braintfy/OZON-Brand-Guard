# OZON Brand Guard

Расширение для браузера — поиск дубликатов и защита брендов на маркетплейсе OZON. Обнаруживает подделки через раздел «Другие продавцы» на страницах товаров.

## Основные возможности

### Поиск дубликатов (v3.x)
- **3 стратегии сбора** — ручной ввод SKU, текущая страница товара, пакетный сбор всех товаров из seller.ozon.ru
- **Автоматический парсинг** — расширение открывает каждую страницу товара и собирает всех продавцов из секции «Другие продавцы» / «Есть дешевле»
- **Пауза / Продолжение** — можно приостановить сканирование и продолжить с того же места (даже после закрытия popup)
- **Остановка** — с сохранением частичных результатов в историю
- **Копирование SKU** — иконка копирования рядом с каждым SKU + массовое копирование
- **Экспорт** — Excel (полный отчёт), SKU Excel (только артикулы нарушителей), CSV
- **Вайтлист** — исключение партнёров по SKU, названию магазина или ИНН
- **История поисков** — последние 10 сессий с возможностью загрузить результаты

### Пакетный сбор SKU
- Автоматическое открытие страницы `seller.ozon.ru/app/products`
- Парсинг таблицы товаров с пагинацией (мульти-стратегия: атрибуты, CSS-классы, computed styles)
- Фильтрация по статусу — пропускает «Не продаётся», «Убран из продажи», «Заблокирован»
- Защита от двойного запуска

### Защита бренда (legacy)
- Сканирование продавцов и товаров через Кабинет бренда
- Автоматическая подача жалоб на нарушителей
- Фильтр по странам (CN, TR, KR, IN)
- Расписание автозапуска
- История жалоб до 5000 записей

## Совместимость

Работает во всех Chromium-браузерах:
- Яндекс Браузер
- Google Chrome
- Microsoft Edge
- Opera / Brave

## Установка

1. Скачайте папку `ozon-brand-guard`
2. Откройте `chrome://extensions/` (или `browser://extensions/` в Яндекс Браузере)
3. Включите **Режим разработчика**
4. Нажмите **Загрузить распакованное расширение**
5. Выберите папку `ozon-brand-guard`

## Быстрый старт — Поиск дубликатов

### Ручной ввод SKU
1. Откройте расширение (синий щит)
2. На вкладке **Дубликаты** выберите стратегию **Ручной ввод SKU**
3. Введите артикулы через запятую, пробел или каждый с новой строки
4. Нажмите **Найти дубликаты**
5. Результаты появятся в блоке «Результаты» и на вкладке **Отчёт**

### Пакетный сбор (все товары)
1. Выберите стратегию **Пакетный сбор (все мои товары)**
2. Нажмите **Найти дубликаты**
3. Расширение само откроет `seller.ozon.ru/app/products`, соберёт все SKU и проверит каждый

### Текущая страница
1. Откройте страницу товара на `www.ozon.ru`
2. Выберите стратегию **Текущая страница товара**
3. Нажмите **Найти дубликаты**

### Пауза и продолжение
- **Пауза** — приостанавливает после обработки текущего SKU. Результаты доступны для просмотра и скачивания
- **Продолжить** — возобновляет с того же места. Работает даже после закрытия и повторного открытия расширения
- **Остановить** — прекращает сканирование, сохраняет частичные результаты в историю

### Экспорт результатов
- **📋 Копировать SKU** — все артикулы дубликатов в буфер обмена
- **📥 SKU Excel** — только артикулы нарушителей одним списком
- **📊 Excel** — полный отчёт с ценами, продавцами и ссылками
- **📄 CSV** — для обработки в любых таблицах
- **📋** (иконка рядом с SKU) — копирование отдельного артикула

## Структура вкладок

| Вкладка | Описание |
|---------|----------|
| **Дубликаты** | Выбор стратегии, ввод SKU, запуск/пауза/стоп, быстрые результаты |
| **Отчёт** | Таблица дубликатов + группировка по продавцам + история поисков + тех. логи |
| **Настройки** | Вайтлист дубликатов, уведомления, экспорт/импорт данных |
| **Бренды** | Legacy — проверка через Кабинет бренда (жалобы на продавцов/товары) |
| **Инфо** | Быстрый старт + советы |
