# OZON Brand Guard

🛡️ Расширение для браузера, которое автоматически защищает ваши бренды от подделок на маркетплейсе OZON.

## Возможности

### Жалобы на продавцов
- **Сканирование продавцов** — автоматический парсинг страницы «Продавцы бренда»
- **Автопоиск брендов** — определение брендов на странице с накоплением по страницам
- **Фильтр по странам** — определение нарушителей по стране (CN, TR, KR, IN)
- **Авто-жалобы на продавцов** — заполнение текста, прикрепление торгового знака, отправка

### Жалобы на товары
- **Сканирование товаров** — парсинг страницы «Список товаров бренда» (`brand-products/all`)
- **Фильтр по Whitelist** — товары своих продавцов пропускаются, на чужие подаются жалобы
- **Отдельные настройки** — свой текст жалобы и файл для товарных жалоб

### Общее
- **Авто-навигация** — расширение само открывает нужную страницу OZON с любой вкладки
- **Whitelist** — разрешённые продавцы по имени или ИНН не получают жалобы
- **Мульти-бренд** — разные шаблоны и файлы для каждого бренда, красная подсветка если файл не прикреплён
- **История жалоб** — до 5000 записей в таблице + экспорт CSV/Excel/TXT
- **Технические логи** — отдельная вкладка с DIAG-логами в реальном времени
- **Расписание** — автозапуск через Chrome Alarms API
- **Dry Run** — тестовый режим без отправки жалоб
- **Экспорт/импорт** настроек

## Совместимость

Работает во всех Chromium-браузерах:
- Яндекс Браузер
- Google Chrome
- Microsoft Edge
- Opera / Brave

## Установка

1. Скачайте папку `ozon-brand-guard`
2. Откройте в браузере `chrome://extensions/` (или `browser://extensions/` в Яндекс Браузере)
3. Включите **Режим разработчика**
4. Нажмите **Загрузить распакованное расширение**
5. Выберите папку `ozon-brand-guard`

## Быстрый старт

1. Нажмите на иконку расширения (синий щит) — с любой страницы
2. Вкладка **Бренды** → «Найти бренды на странице» → отметьте свои
3. Добавьте текст жалобы и файл торгового знака для каждого бренда
4. Вкладка **Главная** → «Сканировать + жалобы» → задержка 15-30с → **Сканировать**
5. Для жалоб на товары: вкладка **Товары** → настройте текст и файл → **Сканировать товары**

> Подробная инструкция: [GUIDE.md](GUIDE.md) | Карта проекта: [project.md](project.md)

## Вкладки расширения

| Вкладка | Описание |
|---------|----------|
| **Главная** | Двойная панель: Продавцы + Товары, задержка, расписание, Dry Run |
| **Настройки** | Бренды, Whitelist, фильтр стран, жалобы на товары, данные |

| **Инфо** | Быстрый старт + советы |

## Настройки

### Бренды
Добавьте бренды через вкладку «Бренды». Для каждого можно задать:
- Название (как на OZON)
- Текст жалобы (или используется текст по умолчанию)
- Файл торгового знака (PDF, JPG, PNG)

> Бренды без прикреплённого файла подсвечиваются красным.

### Товары
Вкладка «Товары» — отдельный текст жалобы и файл для жалоб на конкретные товары. Если не заданы — используются настройки бренда.

### Whitelist
Добавьте разрешённых продавцов по имени или ИНН через вкладку «Whitelist». Используется и для фильтрации продавцов, и для фильтрации товаров.

### Фильтр по странам
Включите/выключите фильтрацию по стране продавца. По умолчанию CN (Китай) помечен как нарушитель.

### Задержка
Между жалобами выдерживается пауза 10-300 секунд (настраивается, рекомендуется 15-30с) + случайный джиттер 0-5 секунд.
