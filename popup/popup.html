Быстрый старт

1
Настройки → Мои бренды

Нажмите «Найти бренды» или добавьте вручную. Для каждого бренда задайте текст жалобы и прикрепите документ ТЗ.

2
Настройки → Whitelist

Добавьте себя и партнёров. Они не получат жалоб ни как продавцы, ни как владельцы товаров.

3
Главная → Продавцы

Выберите «Сканировать» для теста или «+ Жалобы» для боевого режима. Нажмите «Запустить» — страница откроется автоматически.

4
Главная → Товары

Аналогично для товаров. Жалобы подаются на товары чужих продавцов (не в Whitelist).

5
Расписание

Включите автозапуск для продавцов и/или товаров. Расширение само откроет страницу и начнёт работу.

Советы

  • Начните с тестового режима (Dry Run) — убедитесь что нарушители определяются верно
  • Если жалоба на копию карточки (без ТЗ) — включите «Не прикладывать файл» в настройках товаров
  • Файл для товарных жалоб берётся из бренда, если не загружен отдельно
  • Для работы расписания браузер должен быть открыт
