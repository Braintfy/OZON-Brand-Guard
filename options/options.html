OZON Brand Guard

Расширенные настройки

Как пользоваться

  1. Откройте Продавцы бренда в OZON Seller
  2. Нажмите на иконку расширения Brand Guard
  3. Настройте бренды, whitelist и текст жалобы
  4. Выберите режим: «Только сканировать» или «Сканировать + жалобы»
  5. Нажмите «Сканировать»

Совместимость

Расширение работает в браузерах на основе Chromium:

  • Яндекс Браузер
  • Google Chrome
  • Microsoft Edge
  • Opera / Opera GX
  • Brave

Технические детали

Manifest Version: 3
Content Script: seller.ozon.ru/app/brand/sellers
Хранение данных: chrome.storage.local
Планировщик: chrome.alarms
Файлы ТЗ: хранятся в base64 внутри storage

Поддержка

Основные настройки доступны через popup расширения (клик по иконке).

Все настройки можно экспортировать/импортировать через JSON файл в разделе «Настройки» popup.
