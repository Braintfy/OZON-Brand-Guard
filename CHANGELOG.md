# Changelog

## [3.0.0] - 2026-03-15

### Added
- **Поиск дубликатов** — новая главная функция: ввод SKU → парсинг секций «Есть дешевле» / «Другие продавцы» на страницах товаров ozon.ru → сбор SKU конкурентов
- **`content/content-duplicates.js`** — content script для парсинга дубликатов на `www.ozon.ru/product/*` (4 стратегии поиска: виджеты, заголовки, TreeWalker, все ссылки)
- **Вкладка «Дубликаты»** — ввод SKU (textarea), задержка, кнопка запуска, прогресс-бар, быстрые результаты
- **Вкладка «Отчёт»** — полная таблица найденных дубликатов (Мой SKU, SKU дубликата, Название, Цена, Продавец, Ссылка)
- **Копировать все SKU** — кнопка копирования уникальных SKU дубликатов одним списком для поддержки OZON
- **Экспорт Excel/CSV** — полный отчёт дубликатов с ценами, ссылками, продавцами (`duplicates_TIMESTAMP.xls/.csv`)
- **Вайтлист дубликатов** — фильтрация по SKU, названию магазина, ИНН (вкладка «Настройки»)
- **`duplicateWhitelist`** — новый массив в config: `[{value, type: 'sku'|'seller'|'inn'}]`
- **`lastDuplicateResults`** — сохранение результатов между сессиями в `chrome.storage.local`
- **`www.ozon.ru/*`** — добавлен в `host_permissions` для доступа к страницам товаров
- **Плавающая панель дубликатов** — drag-панель на странице OZON с логами и статусом парсинга
- **`launchDuplicates`/`stopDuplicates`** — новые actions в service-worker.js с последовательной обработкой SKU

### Changed
- **Главная вкладка → Дубликаты** — поиск дубликатов теперь основной функционал расширения
- **5 вкладок** — Дубликаты (главная), Отчёт, Настройки, Бренды (legacy), Инфо
- **Вкладка «Бренды»** — весь старый функционал (продавцы, товары, жалобы, расписание, бренды, whitelist) перенесён сюда в рабочем состоянии
- **Уведомление deprecated** — на вкладке «Бренды» оранжевый баннер: «Устаревший функционал. OZON ввёл ограничения на частое использование Кабинета бренда»
- **Настройки** — переработаны: вайтлист дубликатов, общие настройки, данные
- **Инфо** — обновлённые инструкции для поиска дубликатов (3 шага), советы
- **Version** — 2.1.3 → 3.0.0 (major: новый основной функционал)
- **manifest.json `description`** — обновлено: «Поиск дубликатов и защита брендов на OZON»
- **`service-worker.js`** — добавлены: `launchDuplicateSearch()`, `processDuplicateSku()`, `handleDuplicatePageResult()`, `stopDuplicateSearch()`, `duplicateScanState`
- **`popup.js`** — переписан: новые render-функции дубликатов, export Excel/CSV дубликатов, обработчики сообщений
- **`popup.css`** — новые стили: `.notice--info`, `.notice--deprecated`, `.dup-progress*`, `.dup-group*`, `.dup-item*`

## [2.1.0] - 2026-03-12

### Fixed
- **Popup закрывался при навигации** — вся логика навигации/инжекции/запуска перенесена в `background service-worker.js`. Popup отправляет `launchSellers`/`launchProducts` в background, который сам открывает страницу, инжектит скрипт и запускает. Popup может безопасно закрыться
